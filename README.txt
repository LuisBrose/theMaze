Dieses Projekt setzt eine Installation von Node.js voraus https://nodejs.org/.
Es wurde mit Node.js 18.12.0 getestet.

Der Server kann gestartet werden mit bin/www -> rechte Maustaste -> run.
In bin/www können sie auch den Port ändern.

Sie können die API erkunden mit swagger-ui unter http://localhost:3000/api-docs, wenn Sie den Server gestartet haben.

Files im Verzeichnis public werden statisch unter localhost:3000 geliefert. Bitte platzieren Sie Ihren Code in
einem Verzeichnis analog zu studentAundB in public. Ich werde Ihren Code beim Test genau dort entpacken.

Ändern Sie den Code des Servers nicht. Wenn Sie Verbesserungsvorschläge haben, bitte wenden Sie sich an die Betreuer.
